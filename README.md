# AirwayPlanningAPI
Airway Planning API - Spring Boot REST API

This REST API is created using Spring Framework and H2 database. 


GET mappings:

* /flights
* /todaysFlights
* /flights/date/{date}
* /flights/airline/{airlineId}
* /flights/airport/{airportId}
* /airlines
* /airports

POST mappings:

* /flights
* /airlines
* /airports

DELETE mappings:

* /flights/{id}
* /airlines/{id}
* /airports/{id}
